# Observations Présidentielles 2027

Site statique non partisan présentant une analyse factuelle et sourcée des cinq principaux candidats pressentis à l'élection présidentielle française de 2027.

## 🔗 Voir le site

**[https://mlolita26.github.io/observations-presidentielles-2027/](https://mlolita26.github.io/observations-presidentielles-2027/)**

Pages utiles :
- [Accueil](https://mlolita26.github.io/observations-presidentielles-2027/) — les 5 candidats + 11 sujets
- [Aperçu complet (HTML statique)](https://mlolita26.github.io/observations-presidentielles-2027/apercu.html) — toutes les données sur une page
- [Méthodologie](https://mlolita26.github.io/observations-presidentielles-2027/methodologie.html)
- [Glossaire](https://mlolita26.github.io/observations-presidentielles-2027/glossaire-static.html)

## Caractéristiques

- HTML / CSS / JavaScript vanilla. Aucun framework, aucune dépendance backend.
- Toutes les données proviennent de sources publiques (HATVP, CNCCFP, Assemblée nationale, Sénat, Parlement européen) et sont accompagnées d'un lien vers leur origine.
- Présomption d'innocence stricte respectée. Aucune couleur partisane.
- Responsive, accessible, imprimable.

## Lancer en local

```
python -m http.server 8000
```

Puis ouvrir [http://localhost:8000/](http://localhost:8000/).

## Déploiement

Hébergeable sur GitHub Pages, Netlify ou Vercel sans configuration.
